Axios

  • 클라 및 서버사이에 HTTP 요청을 만들고 응답을 처리하는데 사용되는 자바스크립트 라이브러리
  • Promise API 기반으로 비동기 처리를 더 쉽게 처리 가능

비동기 처리의 특성

  • 작업이 시작되는 순서가 아닌 완료되는 순서에 따라 처리됨

    • 비동기 처리의 어려움

  • 코드의 실행순서가 불명확하다는 단점 존재
  • 실행결과를 정확히 예측하며 코드를 작성하기 어려움

    • 비동기 처리 관리 방법
